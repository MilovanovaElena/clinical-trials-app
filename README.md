# clinical-trials-app
Веб-приложение для клинических исследований, позволяющее пользователям вводить данные о самочувствии и принимаемом препарате, выбирать исследование, получать простую оценку состояния и сохранять информацию в базу данных. Проект реализован на Python/Django с использованием PostgreSQL для хранения данных.
